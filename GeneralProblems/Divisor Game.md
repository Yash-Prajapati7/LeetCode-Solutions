Leetcode Question : [Divisor Game](https://leetcode.com/problems/divisor-game/)

### Java

```java
class Solution {
    public boolean divisorGame(int n) {
        if(n % 2 == 0) return true;
        else return false;
    }
}
```

### C++

